# ⚡ IA Audit Pro

AI-powered website audit SaaS — SEO, Performance, Security & Bug analysis.

## Stack
- Frontend: React + Vite → Vercel
- Backend: Node.js + Fastify → Railway
- Database & Auth: Supabase

## Setup

### Backend
```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

### Frontend
```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```
